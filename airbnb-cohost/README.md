## 💡 重要提醒

### ✅ DO (要做的)
- 始终专业和responsive (1小时内回复)
- 超预期服务（小礼物、快速解决问题）
- 记录一切（照片、消息、费用）
- 定期发报告给房东
- 持续优化pricing和listing

### ❌ DON'T (不要做的)
- 不要接ADU（San Jose禁止）
- 不要接Sunnyvale的房源（太严格）
- 不要超过180天限制（会被罚）
- 不要忽视邻居投诉
- 不要承诺不现实的收益

## 📞 常见问题

**Q: 我需要自己的房子吗？**
A: 不需要！你是co-host，管理别人的房子。

**Q: 我需要多少启动资金？**
A: 几乎为零。房东负责房子和清洁费用。你只需要时间和手机。

**Q: 我需要LLC或公司吗？**
A: 不是必须，但建议（税务优惠+责任保护）。可以以个人名义开始。

**Q: 我需要保险吗？**
A: Airbnb提供$3M Host Protection Insurance。你可以考虑额外的liability insurance（~$30-50/月）。

**Q: 180天上限是什么意思？**
A: San Jose规定：如果你（房东）不在家，一年最多只能出租180晚。你需要监控并提前block日历。

**Q: 如果房东违规怎么办？**
A: 在合同里明确责任（见 `legal/co-host-agreement.md`）。你负责提醒，但房东负责最终合规。

**这是一个可行的生意模式！**
