## 🔍 如何查看竞品清洁费？

### 方法1: Airbnb搜索

1. 去Airbnb.com
2. 搜索"Campbell, CA"
3. 选择日期（例：3月28-31，3晚）
4. 筛选：1 bedroom
5. 点进listing
6. 看价格breakdown（点"Show price details"）
7. 记录：每晚房价 + 清洁费

### 方法2: 估算法（如果看不到breakdown）

```
Total price (5晚) = $1,000
假设房价 = $160/晚
$160 × 5 = $800
清洁费估算 = $1,000 - $800 = $200
```

### 方法3: AirDNA (付费工具)

- 订阅AirDNA ($19.99/月)
- 输入地址
- 看附近listings的详细定价
- 包括清洁费数据

## 💰 预期盈亏分析

### Scenario: 1BR Campbell

**假设：**
- 房价：$150/晚
- 清洁费：$180（收客人）
- 清洁成本：$195（付给清洁公司）
- 你的佣金：25%
- 平均入住：3晚

**单次预订：**
```
客人付：
房价: $150 × 3 = $450
清洁费: $180
Total: $630

房东收（扣除Airbnb 3%）:
$630 × 97% = $611

房东支出：
清洁: $195
净收入: $416

你的分成（25%）:
$416 × 25% = $104

清洁费亏损：
收客人: $180
付清洁: $195
亏损: -$15

但这$15从你的$104佣金里出
实际你拿: $104 - $15 = $89
```

**结论：你每次预订赚$89**

---

### 优化方案：提高清洁费到$220

```
客人付：
房价: $150 × 3 = $450
清洁费: $220
Total: $670

房东收: $650
房东支出: $195（清洁）
净收入: $455

你的分成: $455 × 25% = $114

清洁费盈余:
收: $220
付: $195
盈余: +$25

总收入: $114 + $25 = $139
```

**你每次赚$139（多$50）**

**代价：** 客人可能觉得$670/3晚 = $223/晚有点贵
