## 第一次见面（咨询）

### 你要做的事

#### 1. Site Visit（看房）

**去房东家实地看房，带上：**
- [ ] 笔记本/iPad（记录）
- [ ] 卷尺（量房间）
- [ ] 手机（拍照）

**看什么？**
- [ ] 房间大小（几卧室？多大？）
- [ ] 家具情况（有床吗？沙发？桌椅？）
- [ ] 浴室状况（干净吗？需要翻新吗？）
- [ ] 厨房设施（有基本器具吗？）
- [ ] WiFi（测速度）
- [ ] 停车位（有吗？在哪？）
- [ ] 周边环境（安静吗？安全吗？）

#### 2. 合规检查

**问房东：**
- [ ] 这是你的主居（primary residence）吗？
  - San Jose: 必须是
  - Campbell: 不要求
- [ ] 有HOA吗？HOA允许短租吗？
- [ ] 有房贷吗？银行允许短租吗？
- [ ] 房子是ADU吗？
  - 如果是 → ❌ San Jose/Campbell都不允许

#### 3. 收益估算

**当场给房东算账：**

```
例子：Campbell 1BR

预估每晚房价: $150
预估入住率: 50%（一个月15天）
预估月收入: $150 × 15 = $2,250

扣除：
清洁费（房东收）: $200 × 4次 = $800
你的佣金（25%）: $2,250 × 25% = $562
Airbnb服务费（3%）: $2,250 × 3% = $68

房东净收入: $2,250 - $562 - $68 = $1,620/月
（清洁费另算，房东收$800，付清洁公司$780，赚$20）

总计房东拿: $1,620 + $20 = $1,640/月
你拿: $562/月
```

**告诉房东：**
- "第一个月可能只有50%入住率"
- "拿到好评后，入住率会提高到60-70%"
- "旺季（6-7月）可能达到80%"

#### 4. 讨论期望

**问房东：**
- [ ] 你希望多久开始接待客人？
- [ ] 你能接受的最低入住率是？
- [ ] 你介意客人带宠物吗？
- [ ] 你有任何房间/物品是off-limits的吗？
- [ ] 你多久会回来住？（如果是主居）

**为什么重要？**
- Self check-in（你不用亲自去开门）
- 可以远程改密码
- 客人方便

**推荐品牌：**
- August Smart Lock ($200)
- Schlage Encode ($250)
- Yale Assure Lock ($200)

**安装：**
- 房东自己装，或
- 找锁匠（$50-100安装费）

#### 2. 床上用品

**购买清单：**
- [ ] 床单 × 3套（可以轮换）

  - Costco有便宜的
- [ ] 枕头 × 4个
- [ ] 被子/Comforter × 2
- [ ] 床垫保护套 × 1（防污渍）

**总花费：** ~$200-300

#### 3. 浴室用品

**购买清单：**
- [ ] 浴巾 × 6条
- [ ] 手巾 × 6条

### Week 2: 房源美化

#### 1. Deep Clean（深度清洁）

**联系清洁公司：**
- Master Clean Bay Area
- 要求：Deep clean（不是turnover）
- 花费：$250-350（1BR）

**清洁内容：**
- 所有房间吸尘/拖地
- 擦窗户
- 清洁浴室（马桶、淋浴、洗手台）
- 清洁厨房（擦柜子、清洁灶台）
- 清洁冰箱
- 擦所有表面

#### 2. 拍照（最重要！）

**什么时候拍？**
- Deep clean完成后
- 白天（自然光最好）
- 晴天（如果有窗外景色）

**拍什么？（至少10-20张）**

**必拍：**
1. 客厅（从门口拍全景）
2. 客厅（近距离拍沙发）
3. 卧室（从门口拍全景）
4. 床（近距离拍整理好的床）
5. 浴室（马桶、淋浴、洗手台）
6. 厨房（如果有）
7. 外观（建筑或门口）
8. 停车位（如果有）

**加分项：**
9. 窗外景色（如果好看）
10. 后院（如果有）
11. 工作区（书桌+椅子）
12. 细节（咖啡机、植物、装饰）

**拍照技巧：**
- 横着拍（不要竖着）
- 房间收拾干净（没有杂物）
- 打开所有灯
- 打开窗帘（自然光）
- 不要用滤镜（真实最重要）
- 用手机广角模式（让房间看起来更大）

#### 3. 小装饰（可选但推荐）

**让房间更温馨：**
- [ ] 植物 × 2-3盆（$20-40）
- [ ] 装饰画 × 2-3幅（$30-60）
- [ ] 抱枕 × 2-3个（$20-40）
- [ ] 地毯（如果地板冷）（$50-100）
- [ ] 台灯（如果灯光不够）（$30-50）

**总花费：** ~$150-300

## 创建Listing

### 你需要做的

#### 1. 注册房东的Airbnb账户（或用他现有的）

**如果房东没有账户：**
1. 去Airbnb.com注册
2. 用房东的Email
3. 完善Profile（房东的照片+介绍）
4. 添加房东的银行账户（收款）

#### 2. 创建Listing

**点击"Become a Host"**

**Step 1: 基本信息**
- [ ] 房源类型：Entire place / Private room
- [ ] 地址：完整地址
- [ ] 客人数：最多几个人
- [ ] 卧室数
- [ ] 床数
- [ ] 浴室数

**Step 2: 设施**

**必选：**
- [x] WiFi
- [x] Kitchen（如果有）
- [x] Free parking（如果有）
- [x] Self check-in（Smart Lock）
- [x] TV
- [x] Air conditioning / Heating

**加分项：**
- [ ] Dedicated workspace
- [ ] Backyard
- [ ] Pets allowed（如果房东同意）
- [ ] EV charger（如果有）

**Step 3: 上传照片**

- [ ] 上传10-20张照片
- [ ] 选最好看的客厅/卧室照片做封面
- [ ] 调整照片顺序（重要的在前）

**Step 4: 写标题**

**好的标题例子：**
```
✅ "Cozy 1BR near Apple Campus - Self Check-in"
✅ "Bright Studio in Campbell - Free Parking + WiFi"
✅ "Modern Guest Suite - Quiet Neighborhood"
```

**不好的例子：**
```
❌ "Nice place"
❌ "Great Airbnb!!!"
❌ "房间出租"
```

**公式：**
```
[形容词] + [房型] + [位置卖点] + [1-2个设施]

例：
Spacious 1BR in Campbell - Fast WiFi + Parking
```

**Step 5: 写描述**

**模板：**

```
[一句话总结]
Welcome to my [cozy/modern/bright] [1BR/studio] in [Campbell/San Jose]! 
Perfect for [business travelers/couples/solo travelers].

[空间介绍]
The Space:
• [Queen/King] bed with comfortable mattress
• Fast WiFi (perfect for remote work)
• Smart TV with Netflix
• Fully equipped kitchen
• Free parking spot

[位置]
Location:
• 10-minute drive to Apple Park
• 5 minutes to Downtown Campbell
• Close to restaurants and shops
• Quiet and safe neighborhood

[设施]
Amenities:
• Self check-in with smart lock
• Fresh linens and towels
• Coffee and tea provided
• Washer/dryer in unit

[其他]
Other Things to Note:
• I'm available 24/7 for questions
• Check-in: 3pm | Check-out: 11am
• No smoking, no parties

Looking forward to hosting you!
```

**字数：** 200-500字（不要太长）

**Step 6: 设置价格**

**参考竞品定价：**
1. 搜索Campbell附近类似房源
2. 看他们收多少
3. 定个中间价

**例：**
```
竞品1: $140/晚
竞品2: $160/晚
竞品3: $170/晚

你定: $150/晚（中间价）
```

**其他价格：**
- Cleaning fee: $200
- Weekend pricing: +10-20%
- Weekly discount: -10%
- Monthly discount: -20%

**Step 7: House Rules**

**标准规则：**
- [x] No smoking
- [x] No parties or events
- [x] No pets（除非房东同意）
- [x] Quiet hours: 10pm - 7am
- [x] Maximum [X] guests

**自定义规则：**
```
• Please remove shoes inside
• Respect neighbors - this is a residential area
• Take out trash before checkout
• No cooking strong-smelling food
```

**Step 8: Booking Settings**

**推荐设置：**
- How guests book: **Instant Book** ✅
  - 但要求：Government ID + Good reviews
- Advance notice: **Same day** 或 **1 day**
- Availability window: **12 months**
- Trip length:
  - Minimum: **2 nights**
  - Maximum: **28 nights**

**Step 9: Check-in/Check-out**

- Check-in: **3pm - 10pm**
- Checkout: **11am**
- Check-in method: **Smart lock（填写说明）**

**Step 10: Cancellation Policy**

**推荐：** **Moderate**
- 入住前5天取消，全额退款
- 平衡了灵活性和保护

## 上线！

### 发布前最后检查

#### Final Walkthrough

**再去一次房源，假装自己是客人：**

1. **从门口开始**
   - [ ] 能顺利找到房子吗？
   - [ ] 门锁好用吗？
   - [ ] 进门第一印象好吗？

2. **卧室**
   - [ ] 床干净舒适吗？
   - [ ] 床头有插座吗？
   - [ ] 衣柜有衣架吗？

3. **浴室**
   - [ ] 热水好用吗？
   - [ ] 水压够吗？
   - [ ] 有毛巾和洗漱用品吗？

4. **厨房**
   - [ ] 基本器具齐全吗？
   - [ ] 冰箱干净吗？
   - [ ] 咖啡/茶有吗？

5. **客厅**
   - [ ] WiFi连得上吗？速度够吗？
   - [ ] TV好用吗？
   - [ ] 有舒适的坐的地方吗？

6. **其他**
   - [ ] 垃圾桶在哪？有垃圾袋吗？
   - [ ] 温度舒适吗（空调/暖气）？
   - [ ] Welcome Guide放好了吗？

#### 发布Listing

1. **Preview（预览）**
   - 在Airbnb看preview
   - 检查照片、描述、价格
   - 让房东也看一遍

2. **Publish!**
   - 点击"Publish listing"
   - Airbnb审核（24小时内）

3. **审核通过后**
   - Listing上线！
   - 客人可以搜到了
