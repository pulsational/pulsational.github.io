## 新房源上线流程

### Phase 1: 初步评估 (Day 1)
**Checklist:**
- [ ] 与房东电话/视频会议 (30分钟)
- [ ] 确认房源类型 (主居/投资房/extra room)
- [ ] 核实合规性：
  - [ ] 是否主居? (需住满60天/年)
  - [ ] 是否ADU? (ADU不能短租)
  - [ ] 房东是否有business license?
- [ ] 评估收益潜力 (用Airbtics/AirDNA)
- [ ] 发送《合作意向书》

### Phase 2: 房源准备 (Day 2-7)
**实地考察 (或视频walkthrough):**
- [ ] 拍照 (15-25张)
  - 客厅、卧室、厨房、浴室
  - 细节: 床品、洗漱用品、设施
  - 窗外景色
- [ ] 检查必备设施：
  - [ ] WiFi (测速，至少50Mbps)
  - [ ] 烟雾报警器
  - [ ] 一氧化碳报警器
  - [ ] 灭火器
  - [ ] 急救包
- [ ] 记录问题 (需维修/升级的)

**房东准备事项 (发给房东的checklist):**
- [ ] 安装智能门锁 (推荐: August, Schlage Encode)
- [ ] 配置WiFi guest network
- [ ] 购买基础用品:
  - 床单x2套, 浴巾x4条, 洗漱用品
  - 咖啡、茶、糖、调料
  - 纸巾、垃圾袋、洗洁精
  - 欢迎小礼物 (小零食/瓶装水)
- [ ] 申请business license (协助填表)
- [ ] 配置Airbnb TOT税设置

### Phase 3: Listing创建 (Day 3-5)
**撰写描述 (英文+中文):**
```markdown
# 标题公式
[Property Type] in [Neighborhood] | [Key Feature] | [Distance to Landmark]
例: Cozy 2BR in Willow Glen | Walk to Santana Row | Free Parking

# 描述结构
1. Hook (吸引人的开头)
2. Space (房间描述)
3. Guest access (客人可用设施)
4. Neighborhood (周边)
5. Getting around (交通)
6. Other things to note (重要提示)
```

**定价策略:**
- Base price: 周边竞品平均价 × 0.9 (先打折吸引首批评价)
- Weekend: Base × 1.2
- 节假日: Base × 1.5
- 长住折扣: 7天-10%, 30天-20%
- 清洁费: 实际成本 + $20
- 最短住宿: 2晚 (可调整)

**House Rules模板:**
```
✓ Check-in: 3PM - 10PM (灵活)
✓ Checkout: 11AM
✓ Self check-in with smart lock
✓ No smoking
✓ No parties or events
✓ No pets (或允许宠物+$50/night)
✓ Quiet hours: 10PM - 8AM
✓ Maximum guests: [数量]
```

- [ ] 上传照片 (至少15张)
- [ ] 填写所有amenities
- [ ] 设置即时预订 (Instant Book) - 高曝光
- [ ] 添加guidebook (周边餐厅/超市/景点)

### Phase 4: 上线前测试 (Day 6-7)
- [ ] 自己试住1晚 (测试流程)
- [ ] 测试门锁密码
- [ ] 确认WiFi和所有设施正常
- [ ] 拍"欢迎视频"发给房东
- [ ] 设置日历 (block前2周，准备期)
- [ ] 发布listing!

## 日常运营

### 每天必做
**上午 (9AM-12PM):**
- [ ] 查看新预订 (5分钟内回复)
- [ ] 查看新消息 (1小时内回复)
- [ ] 更新日历 (同步多平台)

**下午 (2PM-5PM):**
- [ ] 检查明天check-in (发欢迎消息)
- [ ] 确认清洁安排
- [ ] 调整价格 (根据市场变化)

**晚上 (8PM-10PM):**
- [ ] 回复晚间询问
- [ ] 查看明天待办

### 每周必做
**周一:**
- [ ] 查看本周预订情况
- [ ] 安排清洁团队
- [ ] 更新动态定价

**周五:**
- [ ] 检查周末入住准备
- [ ] 补货采购 (如需要)

### 每月必做
**月初:**
- [ ] 发送上月收益报告给房东
- [ ] 检查180天限制 (还剩多少天)
- [ ] 更新日历 (开放未来3个月)

**月中:**
- [ ] 竞品价格调研
- [ ] 优化listing (更新描述/照片)

```
✓ Check-in: [Date] after 3PM
✓ Check-out: [Date] by 11AM
✓ Self check-in with smart lock (code sent 24hrs before arrival)

Need anything? Just message me!

Jason
```

### T-1天: 发送Check-in指引
**消息模板:**
```
Hi [Guest]!

Your check-in details:

🏠 Address: [Full Address]
🔑 Door Code: [6-digit code]
🅿️ Parking: [Instructions]
📶 WiFi: [Network Name] / [Password]

📍 How to get here:
[Google Maps link]
[Public transit instructions if relevant]

🚪 Entry instructions:
1. Go to [front door/side gate/etc.]
2. Enter code [####] on the keypad
3. Turn handle to open

The place is ready for you! Trash/recycling info and house guide are on the kitchen counter.

Enjoy your stay! 🌟
Jason
```

### Check-in当天
- [ ] 早上再次检查房间准备就绪

- [ ] 等待客人确认到达
- [ ] 发欢迎消息:
```
Welcome! Hope you found everything easily. 
Let me know if you need anything! Enjoy SJ! 😊
```

## Check-out流程

### Check-out前1天
**消息:**
```
Hi [Guest],

Hope you're enjoying your stay! 

Quick checkout reminders for tomorrow (by 11AM):
✓ No need to strip beds or wash dishes
✓ Just leave keys in lockbox and close door
✓ Please turn off lights/AC

Thanks for being a great guest! 🙏
```

### Check-out当天
**上午11:30 (退房后):**
- [ ] 远程检查门锁log (确认已退房)
- [ ] 通知清洁团队进场
- [ ] 等清洁完成后做walkthrough (照片)

**检查清单:**
- [ ] 有无损坏?
- [ ] 东西齐全?
- [ ] 有无遗留物品? (联系客人)

**下午:**
- [ ] 留评价给客人
```
[Guest] was a wonderful guest! Clean, respectful, and great communication. 
Would happily host again. Highly recommend! ⭐⭐⭐⭐⭐
```

## 清洁协调

### 找清洁团队
**渠道:**
- Taskrabbit ($80-120/次)
- Handy
- 本地Facebook群组 ("Bay Area Cleaning Services")
- Nextdoor
- 朋友推荐

**面试问题:**
- 有Airbnb清洁经验吗?
- 多久能到? (理想: 2小时内)
- 价格? (1BR多少?)
- 周末/节假日可以吗?

### 清洁标准 (Checklist发给清洁team)
**卧室:**
- [ ] 换床单+枕套 (洗净叠放柜子里)
- [ ] 吸尘/拖地
- [ ] 擦窗台+镜子
- [ ] 清空垃圾桶

**浴室:**
- [ ] 擦马桶+sink+shower
- [ ] 补充洗漱用品 (洗发水/沐浴露/卫生纸)
- [ ] 换毛巾
- [ ] 拖地

**厨房:**
- [ ] 洗碗+擦灶台
- [ ] 清空垃圾+换袋
- [ ] 擦冰箱外部
- [ ] 补充咖啡/茶/纸巾

**客厅:**
- [ ] 吸尘/拖地
- [ ] 整理沙发+抱枕
- [ ] 擦桌面

**最后:**
- [ ] 拍"清洁完成"照片发我

## 紧急情况处理

### 分类响应

#### 🔴 Level 1: 立即处理 (30分钟内)
- 门锁坏了/密码不work
- 停水/停电
- 暖气/空调坏了 (冬天/夏天)
- 客人被锁在外面

**行动:**
1. 立刻回复客人: "I'm so sorry! Working on it right now."
2. 电话联系维修 (准备24/7 handyman名单)
3. 如无法当天修好 → 提供替代方案:
   - 全额退款 + 帮订酒店
   - 或转移到backup房源

#### 🟡 Level 2: 当天处理
- WiFi不稳定
- 小家电坏了 (咖啡机/微波炉)
- 客人投诉噪音

**行动:**
1. 2小时内回复
2. 当天安排上门
3. 送小礼物道歉 (Uber Eats gift card $20)

#### 🟢 Level 3: 48小时内
- 补货需求
- 客人小建议
- 轻微维护问题

## 评价管理

### 收到好评 (4-5星)
**回复模板:**
```
Thank you so much, [Guest]! We're thrilled you enjoyed your stay. 
You were a wonderful guest and we'd love to host you again! 🌟
```

### 收到差评 (1-3星)
**先别慌！分析原因:**
1. 是什么问题?
   - 清洁? → 换清洁团队
   - 设施? → 立刻维修/升级
   - 沟通? → 改进回复速度
2. 是否合理?
   - 合理 → 诚恳道歉 + 改进
   - 不合理 → 礼貌but factual回复

**回复模板 (问题合理):**
```
Thank you for the feedback, [Guest]. We sincerely apologize for [issue]. 
We've already [action taken] to ensure this doesn't happen again. 
We appreciate you bringing this to our attention.
```

**回复模板 (不合理/恶意):**
```
We're sorry you felt this way. However, [state facts objectively]. 
We strive to provide accurate listings and responsive service. 
We wish you the best in your future travels.
```

### 鼓励好评策略
- 超预期服务 (小礼物/本地推荐)
- 退房当天先留评价给客人 (他们更可能回评)
- 入住第1天check-in: "Everything ok?" (解决问题在差评前)

## 合规监控

### 每月检查
**180天限制监控:**
```
房源A (2025)
Jan: 12天
Feb: 15天
...
Total: 85/180天
剩余: 95天
```

**工具:**
- 用Excel/Google Sheets记录

- San Jose: 每年续费 (~$195)
- 设日历提醒 (到期前30天)

### TOT税检查
- 每月确认Airbnb正确代收10%
- 查看payout明细

### 法规变化监控
- 订阅San Jose city council newsletter
- 加入Airbnb host forum (本地)
