## 🛏️ BEDROOM(S)

### Bed
- [ ] Strip all bedding (sheets, pillowcases, duvet cover)
- [ ] Check mattress for stains/damage
- [ ] Put fresh fitted sheet on mattress
- [ ] Put fresh flat sheet (or skip if using duvet)
- [ ] Put fresh pillowcases (all pillows)
- [ ] Put fresh duvet cover or bedspread
- [ ] Arrange decorative pillows nicely
- [ ] Smooth out wrinkles — make it look hotel-perfect!

### Furniture
- [ ] Wipe down nightstands (top and sides)
- [ ] Wipe down dresser
- [ ] Wipe down headboard
- [ ] Check inside drawers — remove any guest items

### Closet
- [ ] Check for forgotten items (report to me)
- [ ] Straighten hangers
- [ ] Vacuum or sweep closet floor

### Floors & Surfaces
- [ ] Vacuum carpet (or sweep/mop hardwood)
- [ ] Vacuum under bed
- [ ] Dust windowsills
- [ ] Wipe down mirrors
- [ ] Wipe light switches and door handles

### Windows
- [ ] Close and lock all windows
- [ ] Wipe glass if smudged
- [ ] Open curtains/blinds for photos (then leave as preferred)

---

## 🛁 BATHROOM(S)

### Toilet
- [ ] Scrub inside toilet bowl (use toilet brush)
- [ ] Wipe outside of toilet (tank, seat, base)
- [ ] Wipe down flush handle
- [ ] Check for any clogs or issues

### Sink & Counter
- [ ] Scrub sink (remove toothpaste, hair, soap scum)
- [ ] Wipe down faucet and handles
- [ ] Clean mirror (streak-free!)
- [ ] Wipe down countertop
- [ ] Check drain for hair → remove

### Shower/Tub
- [ ] Scrub tub/shower walls (remove soap scum, mildew)
- [ ] Scrub floor of tub/shower
- [ ] Clean showerhead
- [ ] Wipe down shower door/curtain
- [ ] Check drain for hair → remove
- [ ] Hang shower curtain neatly (if applicable)

### Towels & Linens
- [ ] Remove all used towels
- [ ] Put out fresh towels:
  - [ ] 2 bath towels per guest
  - [ ] 2 hand towels
  - [ ] 2 washcloths
  - [ ] 1 bath mat
- [ ] Fold towels nicely or hang on towel rack

### Restocking
- [ ] Toilet paper (at least 2 rolls visible + extras under sink)
- [ ] Hand soap (refill dispenser or replace bar soap)
- [ ] Shampoo (travel-size or refillable dispenser)
- [ ] Conditioner (travel-size or refillable dispenser)
- [ ] Body wash (travel-size or refillable dispenser)
- [ ] Tissues (box on counter)

### Floors & Trash
- [ ] Sweep and mop floor (pay attention to corners)
- [ ] Empty trash can
- [ ] Put in fresh trash bag
- [ ] Wipe down trash can if dirty

---

## 🍳 KITCHEN

### Dishes & Appliances
- [ ] Wash all dishes, pots, pans
- [ ] Dry and put away
- [ ] Run dishwasher if needed (or leave empty)
- [ ] Wipe down dishwasher exterior
- [ ] Clean microwave (inside and outside)
- [ ] Clean stovetop (remove grease, food particles)
- [ ] Wipe down oven exterior
- [ ] Clean refrigerator (inside and outside):
  - [ ] Remove old food
  - [ ] Wipe shelves and drawers
  - [ ] Check expiration dates
- [ ] Wipe down coffee maker
- [ ] Empty coffee grounds (if drip machine)

### Counters & Cabinets
- [ ] Clear all countertops
- [ ] Wipe down counters (remove crumbs, stains)
- [ ] Wipe down backsplash
- [ ] Wipe down cabinet fronts (especially handles)
- [ ] Wipe down sink and faucet (make it shine!)

### Restocking
- [ ] Dish soap (refill or replace)
- [ ] Hand soap at sink
- [ ] Paper towels (at least 1 roll visible + extras)
- [ ] Sponge/dish brush (replace if worn)
- [ ] Coffee/tea (restock if low):
  - [ ] Coffee pods or ground coffee
  - [ ] Tea bags
  - [ ] Sugar/sweetener
  - [ ] Creamer (check expiration)
- [ ] Trash bags (under sink)

### Trash & Floor
- [ ] Empty all trash cans
- [ ] Put in fresh trash bags
- [ ] Take trash bags to outdoor bins
- [ ] Sweep floor
- [ ] Mop floor (especially under table/chairs)

---

## 🛏️ 卧室

### 床
- [ ] 脱掉所有床品 (床单、枕套、被套)
- [ ] 检查床垫是否有污渍/损坏
- [ ] 铺新床单
- [ ] 套新枕套 (所有枕头)
- [ ] 套新被套
- [ ] 整理装饰枕头
- [ ] 抚平褶皱 — 要像酒店一样完美！

### 家具
- [ ] 擦拭床头柜
- [ ] 擦拭梳妆台
- [ ] 检查抽屉 — 移除客人遗留物品

### 地板
- [ ] 吸尘 (或扫地/拖地)
- [ ] 床底吸尘
- [ ] 擦窗台
- [ ] 擦镜子
- [ ] 擦开关和门把手

---

## 🛁 浴室

### 马桶
- [ ] 刷洗马桶内部
- [ ] 擦拭马桶外部 (水箱、座圈、底座)
- [ ] 检查是否堵塞

### 水槽
- [ ] 刷洗水槽 (去除牙膏、头发、皂垢)
- [ ] 擦水龙头
- [ ] 擦镜子 (无水痕！)
- [ ] 擦台面
- [ ] 清理下水道头发

### 淋浴/浴缸
- [ ] 刷洗淋浴墙 (去除皂垢、霉斑)
- [ ] 刷洗淋浴地板
- [ ] 擦淋浴门/浴帘
- [ ] 清理下水道头发

### 毛巾
- [ ] 移除用过的毛巾
- [ ] 放新毛巾:
  - [ ] 每人2条浴巾
  - [ ] 2条小毛巾
  - [ ] 2条洗脸巾
  - [ ] 1条地垫

### 补货
- [ ] 卫生纸 (至少2卷可见 + 水槽下备用)
- [ ] 洗手液
- [ ] 洗发水
- [ ] 护发素
- [ ] 沐浴露
- [ ] 纸巾盒

### 地板
- [ ] 扫地和拖地
- [ ] 清空垃圾桶
- [ ] 换新垃圾袋

---

## 🍳 厨房

### 餐具
- [ ] 洗所有碗、锅、盘
- [ ] 擦干并放回原位
- [ ] 如需要运行洗碗机
- [ ] 擦洗碗机外部

### 电器
- [ ] 清洁微波炉 (内外)
- [ ] 清洁炉灶 (去除油渍、食物残渣)
- [ ] 擦烤箱外部
- [ ] 清洁冰箱:
  - [ ] 移除过期食物
  - [ ] 擦架子和抽屉
- [ ] 擦咖啡机
- [ ] 清空咖啡渣

### 台面
- [ ] 清空所有台面
- [ ] 擦台面 (去除面包屑、污渍)
- [ ] 擦后挡板
- [ ] 擦柜门 (尤其是把手)
- [ ] 擦水槽和水龙头 (要发亮！)

### 补货
- [ ] 洗洁精
- [ ] 水槽旁洗手液
- [ ] 纸巾 (至少1卷 + 备用)
- [ ] 海绵 (如果旧了就换)
- [ ] 咖啡/茶 (如果少了就补):
  - [ ] 咖啡胶囊或咖啡粉
  - [ ] 茶包
  - [ ] 糖
- [ ] 垃圾袋 (水槽下)

### 垃圾和地板
- [ ] 清空所有垃圾桶
- [ ] 换新垃圾袋
- [ ] 把垃圾袋拿到外面垃圾桶
- [ ] 扫地
- [ ] 拖地
