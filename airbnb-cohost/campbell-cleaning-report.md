## 📈 清洁费定价建议

### Host通常怎么定？

**经验法则：**
- 清洁费 = 25%-50% × 每晚房价

**例子：**
```
房价：$150/晚
清洁费：$75-150

推荐：$100-120（在范围中间偏上）
```

### 覆盖成本 + 小利润

**1卧室例子：**
```
清洁公司收费：$145
你收客人：$150-160
你的利润：$5-15（用于管理时间）
```

**2卧室例子：**
```
清洁公司收费：$170
你收客人：$180-200
你的利润：$10-30
```

## ⚠️ 注意事项

### 清洁公司红旗🚩

- 报价远低于市场价（质量堪忧）
- 没有保险（liability风险）
- 没有background check（安全风险）
- 经常late或cancel
- 不回复消息

### 你需要提供

**给清洁公司：**
- 房子钥匙/门锁密码
- 清洁用品位置
- 床单/毛巾储存位置
- 垃圾桶位置
- 特殊要求（不要动某些东西）

**给客人：**
- 准确的清洁费（在listing里显示）
- 清洁标准说明（在House Rules里）

## 💡 省钱技巧

### 长期折扣

- 和清洁公司谈package deal
- 例：每月保底4次清洁，打9折

### 减少清洁频率

- 设置minimum stay = 3晚（减少turnover次数）
- 长住客人（7+ nights）可以不清洁

### DIY部分工作

- 你自己补货（纸巾、洗漱用品）
- 你自己洗床单（如果有洗衣机）
- 清洁公司只做打扫

**能省多少？**
- $20-40/次
